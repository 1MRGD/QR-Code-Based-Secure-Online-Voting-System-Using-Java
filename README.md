# QR Code Based Online Voting System (Java)

A secure web-based online voting system developed using **Java, JSP, Servlets, and MySQL**.  
The system uses **Barcode-based voter authentication** and **QR Code-based voting** to ensure secure, one-person-one-vote elections.

This project is inspired by QR-based voting concepts and fully implemented in **Java Web Technologies**.
